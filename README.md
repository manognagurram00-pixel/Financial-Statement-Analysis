# Financial Statement Analysis

A Python-based application that analyzes financial statements of publicly listed companies by computing key financial ratios and presenting meaningful insights through an interactive dashboard.

## Features
- Retrieve company financial statements
- Calculate key financial ratios (ROE, ROA, P/E, P/B, Debt-to-Equity, Current Ratio, etc.)
- Visualize company performance using interactive charts
- Generate simple rule-based investment insights

## Tech Stack
- Python
- Flask
- yfinance
- Matplotlib
- SQLite

🚧 Currently under development. Additional analysis modules, visualizations, and reporting features are being added.
